Functional Workout

Clone this repo, run the tests, then run the demo to see
a bunch of examples.

functional workout

The code includes Euclidean algorithm generalizations, Fermat’s factoring method, continued fractions, real analysis experiments, a geometry problem-solving module, modules for combinatorics, Church arithmetic, permutation groups and basic polynomial algebra. We also have an example of getting Racket to talk to Apache CouchDB via http.

  $ git clone https://github.com/tedszy/functional-workout.git
  $ cd functional-workout
  $ raco test
  $ racket demo.rkt

Gu Grading Utility

I wrote this C++ program to compute grades and generate
nice looking reports from CSV files. It uses Sheet,
which is a special class for handling column-aligned text,
and Shrdlu, a simple CSV reader.

Generalized euclidean algorithm

November 8, 2015

With the Euclidean algorithm, you can compute GCDs and continued fractions. But with the generalized Euclidean algorithm you can also change radix and compute integer square roots, among other things. Watch how this is done. This module includes the Fermat factorization code I used to design test questions and class problems. Now you can use it too.

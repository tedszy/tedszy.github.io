Rational geometry with racket

March 16, 2016

Points and lines can take you a long way in geometry. Here is all the material covered in the Racket geometry topic, plus more. There are many more problems and studies (mostly projective type stuff) and the code makes use of some Racket features you might not have seen before (match-let).

The article includes all the code that you need too. All you have to do is install Racket and follow along.

Inside-out: mutual recursion vs. function application

June 8, 2016

Recursion can be imagined as repeated function applications turned inside-out. How about mutual recursion? That’s a notoriously difficult idea to visualize, but even that can be understood the same way. Except that we must use functions which return multiple values. That’s the key.

I go through many examples of this analogy: multiplication of complex numbers, Pauli matrices, Fibonacci-like sequences. If you have problems understanding mutual recursion (like everyone does) then you can work through these examples.
